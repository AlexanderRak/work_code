Office 365 — вопросы и ответы

Одинаковы ли приложения Office на ПК, компьютере Mac и мобильных устройствах?

Нет. Приложения Office оптимизируются для наиболее эффективной работы на каждой платформе и каждом устройстве. Наличие и функциональность приложений зависит от платформы и устройства.


Что происходит по окончании срока действия подписки на Office 365?

Как мне продолжить работать со своими документами? Если вы отмените свою подписку или решите не продлевать ее по окончании срока действия, то приложения Office перейдут в режим только чтения, т. е. вы будете иметь возможность просматривать и печатать документы, но не сможете создавать новые или редактировать существующие. Чтобы продолжить пользоваться Office в режиме полной функциональности, можно приобрести новую версию пакета Office, посетив веб-сайты www.microsoftstore.com и https://comfy.ua или обратившись в любой магазин COMFY. Также можно продолжить пользоваться приложениями в режиме полной функциональности, используя более старую версию пакета Office, или бесплатно пользуясь службой Office Online для выполнения простого редактирования.


Что такое OneDrive?

OneDrive — это облачная служба хранения данных, которую можно использовать для хранения документов, фотографий, музыки, видеоматериалов и файлов других типов.

Служба OneDrive обеспечивает доступ к файлам практически с любого устройства и позволяет предоставлять доступ к ним другим пользователям. Все пользователи службы OneDrive бесплатно получают 15 ГБ пространства. При оформлении подписки на Office 365 дополнительно предоставляется еще 1 ТБ пространства в хранилище OneDrive.


Доступны ли документы, сохраненные в OneDrive, в автономном режиме?

Да. Вы можете создать на компьютере каталог, содержимое которого автоматически синхронизируется с содержимым диска OneDrive в облаке. Благодаря этому вы можете работать с файлами в автономном режиме и обеспечить их постоянную синхронизацию.


Что такое «минуты для звонков по Skype»?

Минуты для звонков по Skype можно использовать для звонков на мобильные и стационарные телефоны друзьям и родственникам независимо от наличия у них учетной записи Skype. С каждой активной подпиской на Office 365 для дома, персональный и для студентов предоставляются 60 минут для звонков по Skype в месяц с ежемесячным обнулением неиспользованных минут. Минуты для звонков по Skype можно использовать для звонков со Skype на стационарные и мобильные телефоны, находящиеся в более чем 60 странах (регионах). Звонки по Skype не распространяются на специальные номера, номера с премиум-тарифом и без региональной привязки. Звонки на телефоны доступны только для определенных стран, дополнительные сведения см. на веб-странице www.office.com/information.


Минимальные требования к оборудованию

  • Компьютер и процессор: Процессор 1 ГГц или выше, x86 или 64-разрядный с набором инструкций SSE2
  • Память: 2 ГБ ОЗУ
  • Жесткий диск: 3 ГБ доступного места
  • Разрешение экрана: 1280x800
  • Графика: Чтобы использовать аппаратное ускорение графики, необходима графическая карта с поддержкой DirectX10.
  • Операционная система: Windows 7 с пакетом обновления 1, Windows 8, Windows 8.1, Windows 10, Windows Server 2008 R2, Windows Server 2012, Windows Server 2012 R2, Windows 10 Server
  • Браузер: Internet Explorer 9, 10, 11; Mozilla Firefox 35.x+; Google Chrome 40+; Microsoft Edge
  • Версия .NET: Требуется платформа .NET 3.5; для некоторых функций также может потребоваться установка .NET 4.0 или 4.5 CLR
  • Прочее: Для использования интернет-функций необходимо подключение к Интернету (за него может взиматься плата). Для использования сенсорных функций необходимо устройство, которое их поддерживает. Впрочем, доступ ко всем функциям и возможностям осуществляется также с помощью клавиатуры, мыши и любого другого стандартного или доступного устройства ввода. Сенсорные функции оптимизированы для использования в ОС Windows 10, Windows 8.1 и Windows 8. Некоторые функции продукта и графической системы могут различаться в зависимости от конфигурации системы. Для некоторых функций может потребоваться дополнительное или более совершенное оборудование или подключение к серверу. Учетная запись Microsoft.
