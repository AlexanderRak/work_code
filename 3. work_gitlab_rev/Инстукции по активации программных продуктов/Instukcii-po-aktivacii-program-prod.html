  • Инструкция по установке и активации MS Office 365 Personal

    1. Для установки Microsoft Office, необходимо его загрузить по ссылке

    2. После перехода по ссылке, вам будет предложено войти в учётную запись Microsoft используя свой логин и пароль.

    2.1 Если учётной записи нет, необходимо выбрать пункт «Создать учетную запись»

    3. Вводим созданную или уже существующую учетную запись Microsoft

    4. После того, как вы авторизовались на сайте Майкрософт, необходимо ввести приобретенный ключ активации продукта Microsoft

    Внимание! ключ состоит из 5 групп символов по 5 символов в каждой группе
    Пример: XXXXX-XXXXX-XXXXX-XXXXX-XXXXX

    После ввода ключа нужно выбрать страну и язык для продукта

    5. Далее вам будет предложено установить метод оплаты, временно нужно выключить этот способ чтоб продолжить загрузку. После чего откроется окно подтверждения где нужно нажать кнопку «Далее»

    6. После подтверждения ключа, отроется страница Microsoft для загрузки программы. Нужно перейти в меню «Installs» и выбрать продукт.
    Нажимаем кнопку «Install Office»

    7. В открывшемся окне выбираем «Other option», для выбора языка интерфейса приложений MS Office. В пункте Choose a version по умолчанию установлено значение «Office – 64 bit», его не нужно менять.

    После выбора языка нажимаем кнопку «Install»

    8. После начнётся процесс загрузки приложений. Выбираем любое место на ПК, куда можно сохранить программу и загружаем.
    После завершения загрузки нужно запустить файл. Процесс установки загрузки MS Office начнётся автоматически.
    Дополнительно ничего активировать не нужно. По завершению загрузки, программа будет уже активирована и готова к работе.

  • Смарт Защита +

    Что такое "Смарт Защита+"

    Упал смартфон, разбился экрна… Что делать? Куда обращаться? Сколько это обойдется? Все эти вопросы урегулирует "Смарт защита+". Услуга, с которой Вам не стоит переживать если экран смартфона разобьется, заниматься поисками сервисных центров и переплачивать за ремонт. Более того, даже не нужно выходить из дому, Вашу технику отправит курьер.

    Что входит в услугу?

    1

    Бесплатный ремонт смартфона в случае механического повреждения дисплея (разбития);

    2

    Восстановление устройства после сбоя ПО;

    3

    Замена или чистка динамика/микрофона при выходе из строя;

    4

    Доставка техники курьером (даже выходить из дому не нужно);

    Что делать, если наступил
    страховой случай:

    В случае разбития экрана даже не нужно будет обращаться в магазин!

    1

    Позвонить на горячую линию ЭКТА по номеру 0 800 305 122 и зарегистрировать обращение.

    2

    Следуя подсказкам оператора, отправить устройство курьером в полной комплектации и пакетом документов службой доставки «Новая Почта». Получатель: СКЛАД "ЭКТА Сервис", г. Киев, отделение Новой Почты номер №268. Все отправки осуществляются за счет получателя по безналичному расчёту – ООО «ЭКТА Сервис».

    3

    В течении 5 дней, клиенту придет СМС-сообщение с решением по заявке на ремонт. Получить отремонтированную технику (в течении 14 дней).

    Какие документы потребуются:

    • заявление на ремонт оборудования (высылается оператором в Viber или скачивается с сайта ЭКТА СЕРВИС)
    • оригинал заполненного гарантийного талона на оборудование с печатями магазина
    • копию фискального чека об оплате стоимости оборудования и услуги "Смарт защита"
    • Копия ИНН
    • эксплуатационные документы на оборудование (если такие предусмотрены)
  • Инструкция по установке и активации AdGuard

    1. С устройство нужно перейти по ссылке на сайт

    2. На главной странице сайта нужно выбрать пункт «Скачать»

    3. После загрузки запустить установку приложения.

    Внимание! для Android устройств, в настройках необходимо дополнительно включить пункт «Разрешить установку из этого источника»

    4. После запуска приложения откроется окно для выбор способа установки, выбираем «Быстрая настройка» (для опытных пользователей можно выбрать «Полная настройка»)

    5. Далее нужно следовать инструкция на экране устройства. После чего перейти к активации приложения

    Активация

    1. Заходим в меню «Активировать премиум», для этого кликаем на три точки в правом верхнем углу экрана

    2. Создаём учетную запись (можно использовать вход через Facebook или Google). Учетная запись потребуется для того, чтоб можно было восстановить лицензию продукта. И на второй вкладке «Лицензионный ключ», приобретенный ключ лицензии.

    1. На ваш компьютер нужно загрузить программу по ссылке, согласно разрядности установленной системы (разрядность можно посмотреть в свойствах компьютера)

    Для системы с разрядностью 32 bit

    Для системы с раздрядностью 64 bit

    2. Запустить загруженный файл на компьютере и следовать инструкциям по установке программы на экране.

    3. Провести активацию приложения перейдя в меню «Используйте приобретенный лицензионный ключ»

    4. После проверки ключа активации, следовать инструкциям на эркане для продолжения установки антивируса. По завершению установки, программа будет активирована и готова к работе.

  • Инструкция по активации Windows 10

    1. Запустить ПК на котором уже установлена операционная система Windows 10.
    Ключ лицензии выглядит следующим образом, это 25 значны код, в формате:
    Пример «XXXXX-XXXXX-XXXXX-XXXXX-XXXXX»

    2. После загрузки компьютера, необходимо зайти в «Пуск» далее «Параметры» и выбрать меню «Обновление и безопасность»

    3. Выбираем раздел «Активация»

    4. Далее выбираем пункт «Изменить ключ продукта»

    5. Вводи приобретенный ключ активации

    После ввода ключа, система проверит ключ и зарегистрирует операционную систему. На этом процесс активации завершен.
