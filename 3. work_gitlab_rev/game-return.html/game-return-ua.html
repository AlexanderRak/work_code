Спеціальні умови повернення для геймінг-товарів

Іноді буває так, що ви мріяли про якийсь товар, довго планували покупку, а купивши, зрозуміли, що він не відповідає вашим очікуванням.

Щоб допомогти вам зробити дійсно правильний вибір і насолоджуватися покупкою по повній, ми зробили супер-лояльні умови повернення саме для геймінг-товарів.

Тестируйте товар максимально акуратно, намагайтеся не знімати захисні плівки та ін.

Переконайтеся, що пройшло не більше 2-х днів від дати покупки

Знесіть всі встановлені програми і поверніть заводські настройки

Зверніться в найближчий магазин COMFY

Необхідні документи:
Документ, що підтверджує покупку (касовий чек або видаткову накладну)
Пластикова карта, в разі якщо ви розраховувалися карткою
Документ державного зразка, який засвідчує особу
Геймінг-товар підлягає поверненню, якщо:

• Є в наявності касовий чек (видаткова накладна)

• Збережений товарний вигляд виробу і його комплектуючих: відсутні ушкодження, подряпини, потертості.

• На ноутбуці або ПК ні активований Windows (але збережено встановлене ПЗ)
