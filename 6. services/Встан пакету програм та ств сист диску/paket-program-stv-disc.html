Установка пакета программ это:

  • Установка и настройка мини-пакета необходимых программ
  • Наглядная демонстрация возможностей установленного ПО
  • Подробная консультация по дальнейшей эксплуатации устройства
  • Создание резервной копии операционной системы на внешний носитель

Пакет включает в себя программы:

  • Для просмотра изображений, фото и видео
  • Для работы с текстом и таблицами
  • Браузеры для быстрого доступа в Интернет
  • Мессенджеры для общения
  • Набор необходимых кодеков

1. Специалист COMFY покажет, как работает каждая установленная программа

2. Расскажет о возможностях, которые вы получаете с набором программ

3. Объяснит, какие программы можно будет добавить в дальнейшем

4. Покажет, как скачивать и устанавливать программы самостоятельно

Дополнительная безопасность

COMFY не позволит вам потерять любимые программы или остаться без операционной системы. Для этого наш специалист создаст резервную копию, которая будет сохранена на флешке или другом носителе
