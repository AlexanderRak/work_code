Основные характеристики ноутбука Asus X543

В ноутбуке используется процессор компании Intel − Core i3-7020U. Он трудится в паре с дискретной видеокартой NVIDIA GeForce MX110. Оперативной памяти в Asus Laptop X543 установлено 6Гб, этого вполне достаточно для гейминга, работы с приложениями, просмотра сайтов сети Интернет.

Для хранения, передачи данных, а также для подключения ноутбука к другим внешним устройствам предусмотрены такие интерфейсы:

  • Wi-Fi – стандарт 802.11 ac, модуль благодаря которому можно подключить лэптоп без использования проводов;
  • HDMI – поддерживает передачу многоканальных цифровых аудиосигналов;
  • Bluetooth – уже давно всем привычная беспроводная технология, которая позволяет обмениваться информацией между несколькими устройствами;
  • USB – 2 традиционных разъема 2.0 и один высокоскоростной 3.0;
  • кардридер.

За автономность ноутбука отвечает емкий аккумулятор.
