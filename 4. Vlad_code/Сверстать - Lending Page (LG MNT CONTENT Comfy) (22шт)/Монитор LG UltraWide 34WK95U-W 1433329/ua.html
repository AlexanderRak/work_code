Nano IPS

Новий рівень відображення кольорів

Живі кольори та надчітке зображення з будь-якого ракурсу з дисплеєм Nano IPS

  • DCI-P3 98%

    Широкий спектр кольорів, що відповідає 98% охоплення колірної гами DCI-P3* та 135% колірного простору sRGB – це кращий показник, який є важливим для роботи з зображеннями та цифровими матеріалами високої якості.

  • CIE 1976, DCI-P3, sRGB

    Ще більш якісна передача кольорів

Сертифікований за стандартом UL для Nano IPS матриць монітор LG 34WK95U забезпечує колірну гаму, на 35% ширшу за sRGB 100%.

* Стандарт DCI-P3 розроблено організацією Digital Cinema Initiatives.

Thunderbolt™ 3

Велика потужність та універсальність використання

Thunderbolt™ 3 ідеально поєднується з MacBook та забезпечує надшвидку передачу даних, та заряд з потужністю 85 Вт

  • Надшвидкий обмін даних

    Thunderbolt™ 3* - це у вісім разів вища за USB-3 та вдвічі більша за Thunderbolt™ 2 швидкість, що гарантує швидкісне з’єднання з дисплеями та іншими пристроями.

* Thunderbolt ™ 3 на ПК з більш ранніми версіями Windows Пристрій Thunderbolt ™ 3 сертифіковано для Windows. Якщо комп'ютер з ОС Windows був придбаний до липня 2016 року, переконайтеся, що на ньому встановлені найновіші версії елементів, зазначених нижче: - BIOS / Thunderbolt ™ 3 Driver / Thunderbolt ™ 3 Firmware (якщо є) Щоб переконатися в тому, що Ваша система оновлена, зв'яжіться з виробником вашого ПК. Також, для отримання додаткової інформації, можете відвідати сайт: https://thunderbolttechnology.net/updates
