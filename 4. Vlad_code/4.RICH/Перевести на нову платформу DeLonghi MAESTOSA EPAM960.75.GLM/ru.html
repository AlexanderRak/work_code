Многофункциональный

Создавайте уникальные напитки с дополнительным контейнером Mix. С помощью веничка смешивайте шоколад с молоком, или готовьте кофе со льдом.

СВЕЖИЙ КОФЕ

В 2 контейнера добавляйте 2 разных сорта кофе. Система Tubeless использует все остатки кофе в кофемолке, поэтому для приготовления эспрессо используется только свежемолотый кофе.

КОНТРОЛЬ ТЕМПЕРАТУРЫ

В кофейной машине 2 независимые системы нагрева для идеального контроля температуры: термоблок для приготовления кофе и система нагрева для приготовления молока. Обе работают одновременно.

ТЕХНОЛОГИЯ ONE TOUCH

Одним касанием готовьте идеальный кофе и напитки, в том числе с молоком. Инновационная технология One Touch использует современную электронику с интуитивным управлением для простого и удобного пользования.

РЕГУЛИРУЕМАЯ КОФЕМОЛКА

Конические стальные кофемолки De'Longhi тщательно калибруются и проверяются, чтобы гарантировать вам идеальный помол. Уровень помола можно выставить вручную в соответствии с предпочтениями по вкусу и бленду кофе.

ДВОЙНОЙ ВАРОЧНЫЙ ЦИКЛ

Мощный заварочный блок позволяет еще лучше персонализировать вкус кофе и варить две чашки эспрессо одновременно.

ТЕХНОЛОГИЯ LATTECREMA SYSTEM

Автоматически распределяет идеальную плотность, кремоподобность и температуру ваших любимых кофейных напитков с молоком. А встроенная функция очистки надежно удаляет грязь со всех компонентов.

СИСТЕМА EASY CLEAN

Емкость обеспечена встроенной функцией очистки: после приготовления напитка, горячей водой и паром промываются все части, которые контактировали с молоком. Также эти компоненты можно поместить в посудомоечную машину для более глубокой очистки.
