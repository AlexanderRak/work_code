Аксессуары для легкости укладки

Насадка-диффузор

Равномерно распределяет поток воздуха по большей площади головы, прибавляя объем у корней. Увеличивает объем для прямых волос и создает легкие волны, обеспечивая длительный результат и естественный вид без пушистости для вьющихся волос.

Насадка-концентратор

Тонкая насадка-концентратор точно направляет воздух для быстрой коррекции деталей прически и идеальной укладки, а также отлично подходит для выравнивания волос.
