Жодних пошкоджень від перегрівання

З технологією ThermoShield ваше волосся надійно захищене від перегріву.

Швидке сушіння без перегрівання

На 20%4 швидше сушіння

Потужніший струмінь повітря з 2300 Вт пришвидшує сушіння на 20%4, водночас забезпечуючи надійний захист волосся від пошкоджень.

Потужний двигун

Високоефективний двигун фена створено для професійного застосування. Він забезпечує потік повітря на швидкості 110 км/год5 для швидкого сушіння та ідеальних укладок.

Для ще кращих результатів

Холодний обдув​
Налаштування холодного повітря забезпечує додаткову фіксацію зачіски.

Складана ручка​
Завдяки компактній конструкції зі складаною ручкою фен легко скласти, відкласти на зберігання та взяти з собою в дорогу.

температури та швидкості​
Обирайте ідеальну комбінацію температури та швидкості для вашого волосся завдяки шести режимам, які забезпечують точне керування для персоналізованої укладки.

Аксесуари для легкості укладки

Насадка-дифузор

Рівномірно розподіляє потік повітря по більшій площі голови, додаючи об’єм біля коренів. Збільшує об’єм для прямого волосся та створює легкі хвилі, забезпечуючи тривалий результат і природний вигляд без пухнастості для кучерявого волосся.

Насадка-концентратор

Тонка насадка-концентратор точно спрямовує повітря для швидкої корекції деталей зачіски та ідеальної укладки, а також чудово підходить для вирівнювання волосся.
