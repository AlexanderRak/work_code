Скоротіть час підготовки разом із насадкою для нарізання та натирання

Одна насадка — стільки свіжих ідей! Якщо ви хочете більше рослинної їжі, готувати вдома з нуля чи урізноманітнити страви, цей інструмент стане вашим кухонним порятунком. Швидкий спосіб додати нарізані чи натерті інгредієнти до ваших улюблених салатів, закусок, гарнірів, основних страв тощо.

Перетворіть ваш міксер на потужний пристрій для нарізання

Підготовка продуктів займатиме мінімум часу! Просто прикріпіть насадку-овочерізку KitchenAid до універсального роз'єму будь-якого нашого стаціонарного міксера — і вона змінить ваше уявлення про кухонну рутину. Свіжі інгредієнти без зайвих зусиль — що може бути краще?

Швидкозмінні барабани економлять ваш час

Якщо потрібно порізати або подрібнити овочі по-різному для однієї страви? Жодних проблем. З цією насадкою для міксера просто змініть швидкозмінні барабани. Замінювати леза легко — кілька секунд, і можна переходити до наступного завдання. Виберіть потрібний барабан і вставте його у корпус насадки. Від нарізання огірків до натирання капусти — миттєво.

Не лише салати: деруни, тако з сиром і ще більше

Насадка-овочерізка KitchenAid відкриває безліч можливостей. Три леза з нержавіючої сталі виконують роботу швидко й ефективно. Нарізайте та натирайте овочі для салатів, дерунів чи запіканок, або навіть сир для піци.

  • Барабан для нарізання: рівномірно нарізає овочі, як-от огірки, кабачки чи селеру.
  • Барабан для середнього натирання: ідеально підходить для сиру та твердих овочів, як-от морква.
  • Барабан для грубого натирання: швидко натирає овочі, як-от капусту, для хрусткого салату або страв у стилі стір-фрай.

Безпечне та просте використання

Подвійний штовхач для продуктів — це зручність і безпека. Він забезпечує ідеальну роботу насадки для фруктів та овочів різного діаметра. Великий штовхач має отвір діаметром приблизно 6 см, а маленький — 3,5 см. І головне — це додатковий елемент безпеки. Завжди використовуйте штовхач для подачі інгредієнтів до лез, щоб захистити пальці.

Компактне зберігання насадки

Хоча насадка KitchenAid для овочів — великий помічник, вона займає мінімум місця. У комплекті йде зручний кейс для зберігання, куди можна скласти корпус насадки та 3 барабани.

Швидке очищення

Після завершення роботи всі три леза та штовхач можна безпечно мити в посудомийній машині на верхній полиці. Основний корпус насадки слід помити вручну в теплій мильній воді. Після миття та висушування складіть усе у кейс — і насадка готова до наступного використання.
