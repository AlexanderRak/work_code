Практична комплектація

Холодильна камера об'ємом 75 літрів оснащена 2 скляними полицями, 3 прозорими полицями-балкончиками та спеціальним контейнером для овочів і фруктів, щоб довше зберігати їх свіжими. Крім того, є зручний контейнер для яєць і корисний піддон для льоду.

Морозильна камера об'ємом 7 літрів з 3-зірковим рівнем ідеально підходить для зберігання морозива, замороженої риби та м'яса.
