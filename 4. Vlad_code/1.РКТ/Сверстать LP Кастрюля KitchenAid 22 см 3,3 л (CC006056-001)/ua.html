Універсальний помічник для ваших страв

Маючи об’єм 3,3 літра, ця каструля ідеально підходить для приготування широкого спектра страв: від варіння та тушкування до створення наваристих супів і борщів для двох-трьох осіб.

Завдяки продуманій формі та особливостям матеріалу, їжа готується рівномірно, а її смак залишається насиченим. Кришка, яка щільно прилягає допоможе зберегти вологу та природні аромати страв.
