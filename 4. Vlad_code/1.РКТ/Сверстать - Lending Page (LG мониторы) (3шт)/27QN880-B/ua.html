Персоналізоване робоче місце

Створюй ідеально налаштоване персоналізоване робоче місце та підримуй правильну поставу завдяки удосконаленій підставці Ergo, яка створена спеціально для тих, хто багато часу проводить сидячи за столом.

Компактний дизайн підставки та легке встановлення

Підставка Ergo компактна, займає мало місця на робочу столі та легко встановлюється. Крім того, для забезпечення чистого робочого простору без зайвих кабелів створено рішення USB-C One Cable, яке забезпечує швидку передачу даних і живлення для зарядки ноутбука завдяки лише одному кабелю.
