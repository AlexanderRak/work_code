Вакуумний очищувач обличчя (пори/ акне) FC 41

Прилад для очищення пор вакуумною технологією дбайливо очищає шкіру та робить її здоровою і доглянутою. Підходить для будь-якого типу шкіри. Має три насадки для використання на різних ділянках

  • ефективне очищення пор вакуумом
  • 5 рівнів інтенсивності
  • РК-дисплей

Глибоке очищення

Вакуумна технологія очищення дозволяє ефективно видаляти вугрі, чорні цятки та ороговілі частинки шкіри. Очищувач обличчя зменшує прояви акне, покращує еластичність шкіри та надає обличчю здоровий вигляд

Три насадки

Кругла насадка (велика) для чищення пор на великих областях шкіри, кругла насадка (маленька) для чищення пор на важкодоступних місцях (наприклад, на носі), овальна насадка покращує еластичність і кровопостачання шкіри обличчя, для чищення пор на крилах носа, навколо очей та губ

Зручність використання

Просте керування завдяки перемиканню однією кнопкою. РК-дисплей відображає стан заряду акумулятора та рівні інтенсивності

Практичний дизайн

Ергономічна форма та легка вага (156 г). В комплекті змінні насадки, 10 змінних фільтрів, кришки для фільтра, USB-кабель для зарядки. Літій-іонна акумуляторна батарея приблизно на 90 хвилин роботи (при рівні інтенсивності 5)
