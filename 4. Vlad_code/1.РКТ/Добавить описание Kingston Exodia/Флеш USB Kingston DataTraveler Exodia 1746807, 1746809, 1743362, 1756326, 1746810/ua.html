DataTraveler® EXODIA

Швидке підключення та передача даних для роботи й навчання.

Високошвидкісний накопичувач USB 3.2 Gen 1 забезпечує просте підключення до ноутбуків, ПК, моніторів та іншої цифрової техніки й дозволяє швидко передавати та зручно зберігати документи, музику, відеофайли та інші матеріали.

Переваги DataTraveler Exodia:

  • ● Висока продуктивність USB 3.2;

  • ● Зворотня сумісність з USB 2.0;

  • ● Спеціальне кільце для фіксації;

  • ● Ковпачок для захисту роз’єму;

  • ● 5 років гарантії.

Висока продуктивність USB 3.2

Завдяки технології USB 3.2 Gen 1, DataTraveler Exodia забезпечує простий доступ до цифрових пристроїв та високу швидкість обміну даними — ідеальне рішення для повсякденного використання на роботі, вдома, в школі чи в будь-якому іншому місці.

Зворотня сумісність з USB 2.0

Накопичувач має зворотню сумісність з USB 2.0 та може використовуватись з пристроями, що оснащені USB-портами минулого покоління.

Спеціальне кільце для фіксації

USB-накопичувач DT Exodia оснащений великим кільцем, що легко закріплюється на брелках для ключів, забезпечуючи додаткове сховище даних, яке завжди під рукою.

Ковпачок для захисту роз’єму

Практичний ковпачок забезпечує захист роз’єму USB-накопичувача від пилу та механічних ушкоджень. При роботі з накопичувачем, знятий ковпачок можна закріпити на зворотному боці корпусу та не перейматися про його втрату.

5-річна гарантія

USB-накопичувач DataTraveler Exodia забезпечений 5-річною гарантією та безкоштовною технічною підтримкою виробника.
