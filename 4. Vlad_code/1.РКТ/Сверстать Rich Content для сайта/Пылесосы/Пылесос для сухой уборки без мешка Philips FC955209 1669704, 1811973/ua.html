  • Високоефективний двигун 900 Вт

    Високоефективний двигун потужністю 900 Вт із понад 50 000 об./хв. забезпечує високу потужність всмоктування для ретельного прибирання

  • Захоплює 99,9% пилу

    Насадка TriActive+ та незмінна висока потужність всмоктування забезпечують прибирання 99,9% дрібного пилу*

    * Захоплення 99,9% пилу на твердій підлозі зі щілинами (IEC62885- 2).
  • Технологія PowerCyclone 7

    Технологія PowerCyclone 7 має аеродинамічну конструкцію для зменшення опору повітря і забезпечення незмінної високої потужності всмоктування. Надзвичайно прискорений потік повітря у циліндричному відділенні та унікальні вихідні лопаті ефективно вловлюють пил із повітря.

  • Насадка TriActive+

    Насадка TriActive+ виконує 3 функції чищення за один раз. Вона має більший отвір спереду для всмоктування більшої кількості забруднень, а спеціально створена підошва видаляє пил глибоко з килимів. Повітряні канали та щітки з обох боків насадки захоплюють пил і бруд вздовж стін чи меблів.

  • Компактний дизайн із ручками попереду і вгорі для легкого перенесення

    Компактний та легкий дизайн забезпечує легке зберігання і підіймання пилососа. У конструкції також передбачено верхню і передню ручки для легкого перенесення.

  • Контейнер для пилу можна гігієнічно спорожнювати однією рукою

    Контейнер для пилу, який легко спорожнювати, створено для гігієнічного спорожнення однією рукою для зменшення хмар пилу.

  • Вбудована в ручку м’яка щітка та насадка для меблів

    Вбудована щітка для пилу знаходиться на ручці, тому завжди готова до використання на меблях, пласких поверхнях і оббивці. Насадку для меблів створено для найкращого чищення м’яких меблів, наприклад подушок, диванів та крісел, і навіть видалення шерсті тварин.

  • Система фільтрації Allergy H13 вловлює понад 99,9% дрібного пилу

    Система фільтрації Allergy H13 захоплює понад 99,9% дрібного пилу, зокрема пилок, шерсть тварин та пилові кліщі, тому ідеально підходить для людей, які страждають від алергії. Рівень фільтрації рівний HEPA 13**.

    **Рівні фільтрації визначаються згідно з EN60312-1-2017 та еквівалентні HEPA 13
