Все необхідне поруч

Завдяки комфортному розміщенню ємностей для фруктів та соку процес соковитискання буде легким та зручним. Ви самостійно здійснюєте контроль за наповненням ємностей для соку та вчасно їх замінюєте.

Технологічні рішення

Передові технології віджиму забезпечують повне збереження поживних речовин та аромату овочів та фруктів під час роботи приладу на низьких обертах.

Потужність та зручність

NEOR JUICER H160WT має потужний двигун на 160 Вт, але при цьому працює безшумно. Соковитискач підійде і для використання всією родиною, бо має велику ємність для соку об’ємом 800 мл.
