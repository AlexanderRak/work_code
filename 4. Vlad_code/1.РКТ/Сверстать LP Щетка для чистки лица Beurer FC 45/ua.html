Масаж та очищення з щіточкою для обличчя FC 45

Електрична щітка для приємного масажу обличчя та щоденного догляду. Робить шкіру м'якою та чистою в 4х рази ефективніше, ніж при ручному очищенні. М'яка насадка для будь-якого типу шкіри. Щітка не боїться води, тож ви можете використовувати її у ванній чи під душем.

  • дві швидкості обертання
  • перемикач з підсвічуванням
  • працює від батарейок

Дбайливе та глибоке очищення

Два режими обертання для ефективного та комфортного пілінгу шкіри. Інтенсивний режим для глибокого очищення, та делікатний режим для більш ніжного догляду за шкірою

Насадка високої якості

М'яка насадка-щіточка для щоденного делікатного очищення обличчя. Дуже ніжна та виконана з приємного на дотик матеріалу. Дбайливо чистить шкіру обличчя і забезпечує глибоке очищення пор

Зручне використання

Щітка оснащена кнопкою вимкнення дисплея з підсвічуванням. А коли потрібно буде підзарядити девайс, увімкнеться індикатор низького рівня заряду акумулятора.

Практичний дизайн

Зручний та приємний дизайн. Має підвісне кільце для зручного зберігання. Водостійка (IPX7). Живлення від батарейок, 2Х1,5 В ААА
