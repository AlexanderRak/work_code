Режим Sear для стейків

Готуйте соковиті стейки в режимі Sear: м'ясо
набуває рум'яної скоринки, але зберігає весь
сік завдяки нагріванню пластин до максимальної
температури.

Функція вафельниці

Не забудьте про солодке на десерт — справжні
віденські вафлі з ніжною скоринкою будуть готові
за лічені хвилини завдяки додатковим
пластинам у комплекті.

  • Пластини нагріваються до
    різної температури
    незалежно один від одного

  • Поради і рецепти
    в додатку для смартфона

  • Захист від перегріву
    і автоматичне відключення.

3 режими гриля

Використовуйте контактний гриль для
експрес-приготування, відкритий гриль — для
одноразового приготування декількох
страв, гриль піч — для прожарювання нижньої
і підрум'янення верхньої частини продуктів.

Рівномірний нагрів

Поверхня пластини прогрівається рівномірно
— прожарювання продуктів залишається однаковим
у середині і по краях.

  • Пластини гриля можна
    мити в посудомийній машині

  • Зміна пластин одним
    рухом

  • Антипригарна поверхня
