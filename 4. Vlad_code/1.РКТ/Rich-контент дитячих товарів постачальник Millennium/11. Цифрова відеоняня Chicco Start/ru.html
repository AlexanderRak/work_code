Видеоняня Chicco Start с цифровой технологией FHSS обеспечивает четкую передачу звука и изображения благодаря высокой устойчивости к помехам.


Функция Smart Eco Mode автоматически регулирует работу устройства, экономя энергию в режиме ожидания.


Основные возможности видеоняни Chicco Start:

  • Ночной режим - четкое видео даже в темноте.
  • Яркий 3,2-дюймовый экран с превосходной передачей цветов.
  • Дальность работы - до 200 м* (без помех на открытом пространстве).
  • Длительное время работы от одного заряда.

Гарантия на цифровую видеоняню Chicco Start - 6 месяцев.
