Разъемы и подключения

  • LAN – разъем, которые позволяет соединять несколько устройств в одну локальную сеть;

  • слот для карты PCMCIA (CI+) – в этот слот устанавливается САМ модуль для приема цифрового телевидения стандарта DVB-Т2.
