Змінюй хід гри

LG UltraGear ™ - це потужний ігровий монітор з високоефективними функціями, які допоможуть перемагати в будь-якій грі.

  • Швидкість

    165Гц | 1мс MBR

  • Колір

    HDR10 | sRGB 95%

  • Технології

    Сумісність з NVIDIA® G-SYNC® | AMD FreeSync™ Premium

Частота оновлення 165Гц

Монітор просто створений для максимальної швидкості. 165 Гц та IPS 1 мс (GTG) дозволяють миттєво діяти й при цьому зображення відображаються більш чітко для плавного ігрового процесу.

* Ілюстрації змальовані для кращого розуміння функцій.
* Порівняння частоти оновлення 60 Гц (ліве зображення) та частоти оновлення 165 Гц.

Час відгуку 1мс MBR

Отримуй конкурентну перевагу з функцією 1 мс MBR, що зменшує розмитість зображення та сприяє більш плавному ігровому процесу.

* Ілюстрації змальовані для кращого розуміння функцій.
*При активації функції 1ms Motion Blur Reduction відбувається зменшення яскравості, тому функції NVIDIA® G-SYNC® / AMD FreeSync™/ DAS (Dynamic Action Sync) / Flicker Safe будуть недоступні для одночасного використання.
* Мерехтіння може виникати під час роботи 1мс MBR.

IPS з sRGB 95%

Дисплей IPS з колірним простором sRGB 95% та HDR10 забезпечує виняткову точність передачі кольору і широкий кут огляду, тому поле битви виглядатиме ще більш реалістичнішим.

* Ілюстрації змальовані для кращого розуміння функцій.

Сумісність з NVIDIA G-Sync®

Монітор протестовано NVIDIA та офіційно визнано сумісним з G-Sync®, що в свою чергу забезпечує плавний і швидкий ігровий процес без підвисань та розривів зображення.

* Ілюстрації змальовані для кращого розуміння функцій.
* Порівняння режиму "ВИМК" (ліворуч) та Сумісність з G-Sync®.

Технологія AMD FreeSync™ Premium

З технологією AMD FreeSync™ Premium геймери можуть насолоджуватися плавними рухами без розривів та підвисань в динамічних іграх з високою роздільною здатністю.

* Ілюстрації змальовані для кращого розуміння функцій.
* Порівняння режиму "ВИМК" (ліворуч) та AMD FreeSync ™ Premium.

Ергономічний дизайн

Грай з комфортом та без відволікань завдяки тонкій рамці екрану та можливості регулювати кут нахилу, поворот та висоту монітора.

Динамічна синхронізація дій

Швидко реагуйте на активність ворога, щоб атакувати першими завдяки режиму «Динамічна синхронізація дій» (Dynamic Action Sync). Режим нівелює затримку відтворення зображеня на екрані монітора та синхронізує відчуття в реальному часі в динамічних іграх.

Стабілізація чорного

Режим «Стабілізація чорного» (Black Stabilizer) надає яскравості затемненим ділянкам на моніторі. Це допомогає виявити противника та атакувати першим.

Оптичний приціл

Для ігор з режимом шутера – функція Crosshair («Оптичний приціл»), яка надає фокусування в центрі монітору для влучних пострілів.

* Зображення змодельовані для кращого розуміння функцій. Вони можуть відрізнятися від реальних.
* Наведені вище зображення з лівого боку ілюструють звичайний режим, в якому функції не підтримується.
