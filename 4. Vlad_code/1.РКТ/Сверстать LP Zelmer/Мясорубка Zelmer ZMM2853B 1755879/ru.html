Электрическая мясорубка Zelmer ZMM2853B

Электромясорубка Zelmer ZMM2854В - универсальный помощник на кухне. В комплекте: 2 решетки с разным диаметром, для получения фарша разной плотности, шнековая насадка с двумя терками. Компактный размер мясорубки позволяет хранить ее прямо в кухонном шкафу, не занимая много пространства.

В комплекте 2 решетки
с диаметром:
- 5 мм
- 8 мм

  • С их помощью легко получить фарш разной фракции и плотности. За одну минуту мясорубка выдает 1,5 кг перемолотого мяса.

В мясорубах Zelmer ножи для переработки продуктов производятся из высококачественного металла, который устойчив к корозиям и повреждениям. В модели ZMM 2853В в комплекте двосторонний нож №5. С легкостью справится со всеми видами мяса, нарезая на однородные кусочки, для получения идеального фарша.

  • Насадка-терка для овощей и фруктов. Комплектуется 2 барабанами для шинковки с отверстиями разной формы и размера для измельчения продуктов. С ее помощью можно быстро измельчить/натереть большое количество продукта.

  • Насадка легко устанавливается, подходит для работы не только с сырыми продуктами, но и с некоторыми видами вареных (картофель, свекла, морковь). Главное правильно подобрать насадку.
