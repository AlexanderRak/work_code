Электрическая мясорубка Zelmer ZMM4080B

Электромясорубка Zelmer ZMM4080В - универсальный помощник на кухне. В комплекте: 3 решетки с разным диаметром, для получения фарша разной плотности, насадка для колбас, толкатель. Компактный размер мясорубки позволяет хранить ее прямо в кухонном шкафу, не занимая много пространства.

В комплекте 3 решетки
с диаметром:
- 3 мм
- 5 мм
- 8 мм

  • С их помощью легко получить фарш разной фракции и плотности. За одну минуту мясорубка выдает 2,5 кг перемолотого мяса.

В мясорубах Zelmer ножи для переработки продуктов производятся из высококачественного металла, который устойчив к корозиям и повреждениям. В модели ZMM 4080В в комплекте двосторонний нож №8. С легкостью справится со всеми видами мяса, нарезая на однородные кусочки, для получения идеального фарша.

Насадка для формирования колбасок. Процесс приготовления вкусных домашних колбасок значительно упрощается благодаря данной насадке.

  • Для удобства использования и обеспечения безопасности для проталкивания продуктов в отверстие лотка предназначен специальный толкатель. Изготовлен из качественного пластика.
