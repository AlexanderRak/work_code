Електрична м`ясорубка Zelmer ZMM2854B

Електром'ясорубка Zelmer ZMM2854В - універсальний помічник на кухні. У комплекті: 2 решітки з різним діаметром, для отримання фаршу різної густини. Насадка для ковбас, штовхач, шнекова насадка із двома терками. Компактний розмір м'ясорубки дозволяє зберігати її прямо в кухонній шафі, не займаючи багато простору.

У комплекті 2 решітки
з діаметром:
- 5 мм
- 8 мм

  • З їхньою допомогою легко отримати фарш різний фракції та щільності. За одну хвилину м'ясорубка видає 1,5 кг перемеленого м'яса.

У м'ясорубах Zelmer ножі для переробки продуктів виробляються з високоякісного металу, який стійкий до корозій та пошкоджень. У моделі ZMM 2854В в комплекті двосторонній ніж №5. З легкістю впорається з всіма видами м'яса, нарізаючи на однорідні шматочки, для отримання ідеального фаршу.

  • Насадка-терка для овочів та фруктів. Комплектується 2 барабанами для шатківниці з отворами різної форми та розміру для подрібнення продуктів. З її допомогою можна швидко подрібнити/натерти велику кількість продукту.

  • Насадка легко встановлюється, підходить для роботи не тільки з сирими продуктами, але й деякими видами варених (картопля, буряк, морква). Головне правильно підібрати насадку.

  • Насадка для формування ковбасок. Процес приготування смачних домашніх ковбасок значно спрощується завдяки цій насадці.

  • Для зручності використання та забезпечення безпеки для проштовхування продуктів в отвір лотка використовуйте спеціальний штовхач. Виготовлений із якісного пластику.
