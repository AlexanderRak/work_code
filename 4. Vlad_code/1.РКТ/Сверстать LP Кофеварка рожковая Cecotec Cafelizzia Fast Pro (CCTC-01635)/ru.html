Попробуйте новый вкус кофе вместе с CECOTEC Cafelizzia Fast Pro. Нежный американо, насыщенный эспрессо – выберите напиток по вашему вкусу. Система Force Aroma обеспечит нежную пенку и приятный аромат.

Высокая мощность

Благодаря мощности 1350 Вт ваш любимый напиток будет приготовлен за считанные секунды. Раскройте новые вкусовые впечатления вместе с CECOTEC.

Функция Thermoblock

Данная система обеспечивает поддержание нормального температурного режима для приготовления вкуснейшего кофе.

Насос с поразительной мощностью

Приготовьте невероятно вкусный напиток максимально быстро, ведь кофеварка имеет насос с давлением 20 бар. Очень приятный аромат создается с помощью системы Force Aroma.

Нежная пенка

Благодаря насадке пароварки можно легко взбивать молоко и создавать нежную пенку с возможностью регулировки процесса. Наслаждайтесь ароматным напитком у себя дома.

Вкусные горячие напитки

Кофеварка также подходит для приготовления чая, глинтвейна и других горячих напитков, открывая для вас новые вкусовые оттенки.

Одновременное приготовление двух порций кофе

Двойное выпускное отверстие и два фильтра позволяют готовить одну или две чашки любимого напитка одновременно.

Объемный резервуар

Емкость для воды на 1.2 л легко снимается и очищается, обеспечивая достаточный объем для приготовления ароматного кофе.

Поддон для подогрева посуды

Нагрейте посуду на поддоне из нержавеющей стали перед приготовлением, чтобы кофе был еще вкуснее.
