Апаратний манікюр та педикюр вдома

Бездротовий прилад MP 84 для комфортного та ефективного манікюру та педикюру з десятьма професійними насадками. Легко підстригайте, підпилюйте та поліруйте нігті, видаляйте ороговілу шкіру, кутикулу, мозолі та натоптиші. Забезпечте собі ідеальний догляд за нігтями, руками та ногами швидко у зручний час

  • світлодіодна підсвітка
  • 3 рівні швидкості
  • потужний літій-іонний акумулятор

Комплексний догляд

Десять високоякісних насадок для догляду за нігтями та шкірою: сапфіровий конус, сапфірові диски (м’який та жорсткий), сапфірова пилка для кутикули, факелоподібна насадка, циліндрична насадка, фетровий конус, сапфірова насадка, голчаста насадка, кристалічна насадка для кутикули

Бездротовий та потужний

Потужний літій-іонний акумулятор забезпечує до 2 годин безперервної роботи. Практичний чохол для зберігання також виконує функцію зарядної станції. Ідеальний варіант для подорожей

Зручність використання

Яскраве світлодіодне підсвічування для комфортного використання. Три рівні швидкості до 4400 об/хв. Обертання за годинниковою та проти годинникової стрілки

Практичний дизайн

Елегантний бірюзовий дизайн та зручна ергономічна форма. В комплекті захисна кришка від абразивного пилу. Оснащений роз'ємом micro USB для легкого підключення.
