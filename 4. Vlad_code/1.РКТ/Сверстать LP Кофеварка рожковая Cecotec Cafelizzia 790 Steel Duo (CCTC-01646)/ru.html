Чтобы не греть целый чайник для нескольких чашек чая, можно использовать кофеварку для подачи горячей воды. Это куда быстрее и проще.

В комплекте есть фильтр, который создает алюминиевые капсулы, благодаря чему готовить кофе проще.
Резервуар для воды
