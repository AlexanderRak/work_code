Wi-Fi без мертвых зон

Deco M3W – это дополнительный модуль для Mesh Wi-Fi системы Deco. Расширяйте беспроводную сеть Wi-Fi дополнительной площади 140 кв. м, чтобы устранить зоны со слабым сигналом.

Работает со всеми
модулями Deco

Примечание: для расширения покрытия Deco M3W нужно подключить к модулю Deco.

Удобное подключение

Компактный модуль Deco M3W легко подключается к настенной розетке. Просто добавьте дополнительные модули, подключив их к розеткам в холле, на кухне, в гараже или в подвале.

Подключите усилитель сигнала между модулем Deco и зоной со слабым сигналом. Индикатор сигнала Deco M3W поможет быстро отыскать подходящее место для установки.

  • Горит белым:
    Подключено, сеть доступна

  • Горит красным:
    Подключено, сеть недоступна

    Мигает красным:
    Не подключено

Бесшовный роуминг с одним именем сети

Технология TP-Link Mesh предусматривают совместную работу всех модулей Deco и формирование единой бесшовной сети с одним именем. Смотрите потоковое видео, загружайте большие файлы, играйте в сетевые игры и передвигайтесь по дому без переключения между модулями.

Родительский контроль

Контролируйте доступ детей или персонала к беспроводной сети. С помощью приложения Tether вы сможете блокировать нежелательный контент, ограничить время подключения и даже настроить черные или белые списки для ограничения доступа.

  • Создайте профиль для каждого члена семьи и установите временные ограничения.

  • Одним нажатием обеспечьте безопасность в интернете с помощью настроек, которые блокируют веб-сайты в зависимости от возраста.

  • Установите ограничения на количество времени, которое каждый член семьи может проводить в сети.

  • Узнайте, какие сайты посещают ваши дети и сколько времени они проводят на каждом.

  • Отдохните от Wi-Fi – просто нажмите кнопку, чтобы приостановить доступ в интернет.
