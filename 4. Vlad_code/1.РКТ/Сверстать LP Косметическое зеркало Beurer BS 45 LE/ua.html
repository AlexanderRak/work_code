Косметичне дзеркало BS 45 LE

Настільне косметичне дзеркало з підсвічуванням в стильному чорному дизайні. Велика дзеркальна поверхня, яка обертається у будь-якому напрямку. Додаткове дзеркало з 5-кратним збільшенням

  • яскраве підсвічування
  • сенсорний датчик на поверхні дзеркала
  • практичний чохол для зберігання

Світлодіодне підсвічування

Підсвічування з 28 світлодіодами створює яскраве, рівномірне освітлення для детального відображення. Додаткове магнітне дзеркало зі збільшенням забезпечує ще більшу точність

Зручність користування

Збільшена дзеркальна поверхня (17,5 x 17,5 см) для повного охоплення зображення. Сенсорний датчик на поверхні дзеркала для легкого керування. Після 15 хвилин автоматично вимикається

Налаштування яскравості та положення

Плавне регулювання яскравості дозволяє підібрати найзручніший рівень світла. Дзеркало легко обертається у всі напрямки завдяки кульовому шарніру. Його можна легко закріпити в потрібному положенні

Практичний дизайн

Невелика чаша-підставка дозволяє зберігати корисні дрібнички (косметичні засоби чи ювелірні вироби). Практичний чохол для зручного зберігання. Працює від батарейок, 3 АА х 1.5 В
