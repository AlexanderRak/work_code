Освещение вытяжки будет включаться автоматически, если кто-то будет приближаться.

Конструкция металлических фильтров позволяет легко производить мойку и очистку.

Вытяжка будет автоматически запускаться на второй скорости для оптимизации дальнейшего управления.

Для эффективной оптимизации потребления электроэнергии, легкой очистки и снижению шума, была разработана инновационная система всасывания воздуха по периметру, но не только по центру, а и по ее боковым частям.

Режим рециркуляции воздуха позволяет очищать воздух с помощью угольных фильтров и снова возвращать его в кухню.

Сенсорное управление подчеркивает современный дизайн и значительно облегчает повседневный уход.
