Аксесуари в комплекті

Велика чаша 3,9 л

Чаша з нержавіючої сталі підходить для обробки як для невеликої кількості інгредієнтів, так і обробки інгредієнтів для звичайного тіста (до 2,7 кг) та дріжджового (1 кг).

Приготувати смузі, шейк або коктейль в спекотний день тепер зовсім не проблема з блендером у комплекті.

М'ясорубка ідеально підходить для готування рибних та овочевих паштетів, ковбасних виробів та м'ясного фаршу різних видів.
