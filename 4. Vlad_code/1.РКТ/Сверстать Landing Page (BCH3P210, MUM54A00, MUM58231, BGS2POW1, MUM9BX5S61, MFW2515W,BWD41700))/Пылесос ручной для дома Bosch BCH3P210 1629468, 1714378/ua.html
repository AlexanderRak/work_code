Пилосос 2-в-1

Акумуляторний пилосос та ручний в одному приладі для більш легкого та зручного прибирання.

Два режими роботи

Режим 1: для "легкого" прибирання; максимальний час роботи

Режим 2: для "звичайного" прибирання; середні витрати часу.

Щітка AllFloor Brush

Моторизована електрощітка AllFloor (до 3500 об / хв) забезпечує результати очищення на всіх типах поверхні, включаючи паркет, килими та плитку. Легко виймається та очищується від сміття.

Система RobustAir – це постійно висока продуктивність

Система RobustAir забезпечує високу продуктивність прибирання, навіть коли контейнер для пилу заповнений.
