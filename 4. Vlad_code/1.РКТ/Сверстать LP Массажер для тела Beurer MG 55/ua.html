Масажер для тіла MG 55

Ідеальний варіант для спокійних вечорів після важкого дня або тижня. Портативний ручний масажер з інфрачервоним обігрівом, який забезпечує глибоке розслаблення всіх м’язів.

  • Плавне регулювання інтенсивності
  • Функція ІЧ світла
  • Три змінні насадки

Поплескуючий масаж

Цей масажер робить інтенсивні поплескуючі рухи, які ефективно впливають на глибокі м’язи, чудово знімають напругу та відновлюють м’язовий тонус. А завдяки інфрачервоному теплу м’язи розслабляються швидше.

Індивідуальний масаж

Регулюйте швидкість залежно від власних потреб. Легко та точно обирайте оптимальний рівень інтенсивності.

Змінні насадки

В комплекті ви знайдете три змінні насадки:

  • з круглими виступами — для інтенсивного точкового масажу;
  • округла — для м’якого масажу;
  • циліндрична — для глибокого масажу.

Масажер оснащений поворотним замком для надійної фіксації.

Ергономічний дизайн

Масажер для тіла MG 55 оснащений нековзною ручкою та має легку вагу — менш як 1 кг. Завдяки ергономічній формі їм дуже зручно користуватися самостійно на різних ділянках. Працює від мережі.
