Масажна пілінг щітка для тіла FC 55

Електрична масажна щітка зі знімною ручкою. Лагідний приємний масаж шкіри та м’який ефективний пілінг тіла. Зручно використовувати самостійно по всьому тілу завдяки ергономічній формі. Дві знімні насадки: для всіх типів шкіри та для глибокого пілінгу

  • водонепроникна (IPX 7)
  • двоступеневе обертання
  • потужний літій-іонний акумулятор

Ретельне очищення

Дві насадки-щіточки для комплексного догляду. Більш м’яка насадка підходить для всіх типів шкіри та ідеальна для щоденного догляду. Щітка для пілінгу для м’якого видалення дрібних клітин шкіри та очищення пор

Режими роботи

Два режими швидкості: помірна та швидка. Доступні два типи обертання: кругове для ніжного очищення та осцилювальне для більш глибокого пілінгу

Зручне використання

Насадки щітки легко знімаються. Світлова індикація для відображення швидкості та рівня заряду. Потужний літій-іонний акумулятор, 60 ​​хвилин роботи та 2 години зарядки

Практичний дизайн

Знімна ручка ергономічної форми. З ручкою зручно для спини та важкодоступних зон. Без ручки більш інтенсивне точкове очищення. Приємний масаж та дієвий пілінг в душі та ванній
