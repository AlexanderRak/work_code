Функція високого піднімання
дозволяє легко виймати маленькі скибки хліба.
Вбудована рамка для підігрівання булочок
улюблених булочок, тістечок та рогаликів.
Зовнішня стінка, що не нагрівається
Завдяки термодизайну зовнішні стінки не нагріваються і безпечні для дотику.
Знімний лоток для крихт
Легко чистити завдяки знімному лотку для крихт.
Режим розігрівання дозволяє підігріти попередньо підсмажений хліб за лічені секунди одним натисненням кнопки.
Дизайн із довгим слотом
Один довгий слот підходить практично для будь-якого виду хліба, який ви обираєте.
8 налаштувань підсмажування
дозволяють підсмажувати різні види хліба без ризику підгоряння.

Функція підігрівання за мить підігріває тости, а функція розморожування дозволяє підсмажувати заморожений хліб.
Кнопка скасування миттєво припиняє приготування тостів у будь-який час для безпечної та точної роботи.
