Якісний прилад для манікюру педикюру MP 41

Набір для апаратного манікюру та педикюру з сімома насадками. Самостійний комплексний догляд за нігтями, пальцями рук та ніг. Дозволяє легко отримати професійний результат вдома

  • 2 налаштування швидкості
  • світлодіодна підсвітка
  • чохол для зберігання в комплекті

Комплексний догляд (7 насадок)

Два сапфірові диски (м'який та жорсткий), сапфіровий та фетровий конуси, циліндрична, факелоподібна та шліфувальна насадки. Зручне та ефективне підпилювання, полірування та шліфування нігтів, видалення ороговілої шкіри та мозолів

Налаштування режимів

Дворівневе регулювання швидкості та обертання за годинниковою стрілкою / проти годинникової стрілки з кількістю обертів 3800 / 4600 об/хв. Для індивідуальних налаштувань залежно від насадки та ділянки шкіри й нігтів

Зручність використання

Світлодіодне підсвічування забезпечує хорошу видимість під час використання. Кнопки для регулювання швидкості розташовані на самому приладі для зручності керування

Практичний дизайн

Насадки з довговічним покриттям із сапфіру та фетру. В комплекті зручний чохол для зберігання. Прилад працює від мережі
