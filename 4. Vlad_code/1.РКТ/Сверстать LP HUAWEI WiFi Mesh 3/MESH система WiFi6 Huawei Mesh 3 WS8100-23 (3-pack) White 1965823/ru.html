Технология HUAWEI HomeSecTM для защиты сети

Оснащён операционной системой HarmonyOS, HUAWEI WiFi Mesh 3 защищает домашнюю сеть от атак методами полного перебора и подделки ARP-информации, предоставляет протокол безопасности WPA3 и функцию моментального отключения посторонних.
