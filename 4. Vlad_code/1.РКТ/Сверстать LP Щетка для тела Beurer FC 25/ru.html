Очищение и нежный массаж с щеткой для тела FC 25

Косметическая водонепроницаемая щетка для нежного и тщательного очищения тела. Чистая и гладкая кожа и приятный массаж. Очень удобно использовать самостоятельно благодаря удлиненной форме ручки. Универсальная насадка для всех типов кожи и насадка для мягкого пилинга

  • водонепроницаемая (IPX 7)
  • двухступенчатое вращение
  • 2 уровня скорости

Тщательное ощищение

Две сменные вращающиеся насадки для комплексного ухода. Насадка для ежедневного очищения и для деликатного пилинга. Проводить пилинг лучше в душе или принимая ванну

Режимы работы

Можно выбирать между медленным и быстрым режимами вращения. Двухступенчатое вращение: круговое для деликатного очищения, осциллирующее обеспечивает глубокий пилинг кожи

Удобное использование

Съемные насадки щетки легко очищаются. Простое управление с помощью кнопки. Световая индикация показывает выбранный режим работы

Практичний дизайн

Зручна форма ручки для комфортного самостійного використання. Легко дістатися до важкодоступних зон, таких як спина або сідниці. Працює від батарейок, в комплекті 4шт AA
