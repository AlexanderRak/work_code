Очищення та ніжний масаж з щіткою для тіла FC 25

Косметична водонепроникна щітка для ніжного та ретельного очищення тіла. Чиста і гладенька шкіра та приємний масаж. Дуже зручно використовувати самостійно завдяки подовженій формі ручки. Універсальна насадка для всіх типів шкіри та насадка для м'якого пілінгу

  • водонепроникна (IPX 7)
  • двоступеневе обертання
  • 2 рівні швидкості

Ретельне очищення

Дві змінні обертові насадки для комплексного догляду. Насадка для щоденного очищення та для делікатного пілінгу. Проводити пілінг краще в душі або приймаючи ванну

Режими роботи

Можна обирати між повільним і швидким режимами обертання. Двоступеневе обертання: кругове для делікатного очищення, осцилювальне забезпечує глибокий пілінг шкіри

Зручне використання

Знімні насадки щітки легко очищаються. Просте управління за допомогою кнопки. Світлова індикація показує обраний режим роботи

Практичний дизайн

Зручна форма ручки для комфортного самостійного використання. Легко дістатися до важкодоступних зон, таких як спина або сідниці. Працює від батарейок, в комплекті 4шт AA
