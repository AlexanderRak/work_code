ЗАРЯЖАЕТ ШЕСТЬ ДЕВАЙСОВ ОДНОВРЕМЕННО

Современная семья редко ограничивается использованием только одного девайса. Зачастую дома есть пять-шесть устройств: смартфон, планшет, наушники, часы и другие гаджеты. Но для их зарядки требуется несколько путающихся между собой кабелей, свободных розеток или сетевых адаптеров. ReVolt A6 решает эту проблему, заряжая шесть гаджетов одновременно.

ЗАРЯДКА ДЛЯ ЛЮБОГО ГАДЖЕТА

С помощью док-станции ReVolt A6 вы сможете заряжать девайсы с максимальной скоростью. Для этого на устройстве имеется четыре порта «быстрой зарядки»: два порта PD Type-C 20 Вт и два порта QC USB – 18 Вт. Для удобства в комплекте с зарядкой вы получите два кабеля Type-C/Type-C и два кабеля Type-C/USB. Стоит лишь подобрать провод для конкретного устройства.

БЕСПРОВОДНЫЕ ПРЕИМУЩЕСТВА

Конструкция зарядной станции продумана до мелочей: помимо четырех портов, здесь вы обнаружите два беспроводных интерфейса. Один из них – встроенная зарядка для Apple Watch, второй –беспроводная 10 Вт зарядка для устройств, поддерживающих современный стандарт Qi. Достаточно лишь положить гаджет на специальную контактную площадку.

УМНАЯ ОРГАНИЗАЦИЯ ПРОСТРАНСТВА

Времена, когда нужно было искать девайсы у розеток в разных комнатах, остались в прошлом. Благодаря конструкции ReVolt A6, теперь все гаджеты можно заряжать в одном месте. Одна зарядная станция для шести различный устройств - лучший способ удобно организовать ваше пространство.

СТИЛЬНЫЙ ДИЗАЙН

ReVolt A6 в корпусе черного цвета из металла и закаленного стекла не только сэкономит количество свободных розеток, но также будет отлично смотреться в любом интерьере, благодаря своему современному лаконичному дизайну.
