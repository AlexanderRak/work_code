Швидкий та зручний щоденний догляд
Фени Philips серії 3000 – ідеально підходять для щоденного використання. Інноваційна насадка Thermoprotect зробить сушіння делікатним для будь-якого типу волосся. Висока ефективність фену полегшить стайлінг та скоротить час сушіння. Він компактний та зручний для щоденного використання.
Сушіння при низькій температурі піклується про волосся.
Забудьте про перегрівання вашого волосся, подбайте про його гарний стан. Фени Philips серії 3000 з насадкою ThermoProtect здатні понизити температуру до 15°C водночас швидко висушуючи та роблячи укладку.
* Зниження температури на 15°C можливе при потужності 2100W, максимальній температурі нагріву та максимальній швидкості.
Як працює насадка Thermoprotect?
Тепле повітря дує через отвори насадки ThermoProtect, яка індукує холодне повітря довкола. Врешті тепле повітря змішується з холодним, понижуючи температуру сушіння на 15°С, при використанні фену на потужності 2100 Вт, роблячи можливим делікатне сушіння.
