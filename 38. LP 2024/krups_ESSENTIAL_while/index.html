Система автопромивання

Наша унікальна система автоматично промиває всі необхідні елементи після кожної чашки та гарантує, що всередині не буде залишків кави й накипу. Це забезпечить тривалу роботу кавомашини без поломок.

Закритий заварювальний блок

  • Автоматична система промивання для легкого та швидкого користування.
  • Комфортне використання: без втручання в технологію.
  • Жодного клопоту: машина сама попередить,коли необхідне промивання.
  • Процес промивання відбувається за дуже високої температури води (що неможливо зробити вручну).

Автоматизована система промивання 3 рази на рік

У Krups немає ручного обслуговування. Промивання виконується за запитом вашої машини, чим забезпечується оптимальний рівень гігієни завдяки високотемпературним процесам, які ідеально видаляють кавову олію й залишки вичавків, не пошкоджуючи машину, та економлять ваш час.

Комфорт використання

Сподобався рецепт?

Зручний режим пам'яті забезпечує зручний процес, що дозволяє зберігати два різні рецепти напоїв, зберігаючи ваші оптимальні налаштування кави для ідеальних результатів що разу.

Великий контейнер для зерен

Таблетки для промивання XS300010

  • Продовжують термін безперебійної роботи кавомашини.
  • Промивають кавомашину від залишків кави.
  • Забезпечують ідеальний смак ваших напоїв.

Для видалення накипу F0540

  • Продовжує термін безперебійної роботи кавомашини.
  • Видаляє накип на основних вузлах кавомашини.
  • Покращує смак ваших кавових напоїв.

Картридж для фільтрації води F08801

  • Продовжує термін безперебійної роботи кавомашини.
  • Очищує воду від хлору та небезпечних сполук.
  • Захищає кавомашину від вапняного нальоту.

Набір Все в одному XS530010

  • 2 пакетики засобу для видалення накипу.
